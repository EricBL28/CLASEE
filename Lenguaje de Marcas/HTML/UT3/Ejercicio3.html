Glosario de Términos en Desarrollo de Aplicaciones Multiplataforma

  • Java
    • Lenguaje de programación orientado a objetos ampliamente utilizado en el desarrollo de aplicaciones multiplataforma.
    Android Studio
    Entorno de desarrollo integrado (IDE) oficial para el desarrollo de aplicaciones Android.
    XML
    Lenguaje de marcado utilizado para definir la estructura de los datos en aplicaciones.
    SQL
    Lenguaje estándar para gestionar bases de datos relacionales.
    Multiplataforma
    Desarrollo de software que puede ejecutarse en múltiples sistemas operativos o dispositivos, como Android, iOS, Windows o Linux.
    Kotlin
    Lenguaje de programación moderno y conciso que se utiliza como alternativa a Java en el desarrollo de aplicaciones Android.
