Glosario de Términos Informáticos

A | B | C | D | E | F | H | I | L | M | P | R | S | T | U | V

A

Algoritmo
Conjunto de instrucciones para resolver problemas.
API
Interfaz de programación de aplicaciones.
ASCII
Código estándar de caracteres basado en el alfabeto inglés.

B

Backup
Copia de seguridad de datos.
Big Data
Conjunto de datos masivos que requieren técnicas avanzadas de análisis.
Binario
Sistema numérico que solo utiliza 0 y 1.

C

Cache
Memoria rápida que almacena datos temporalmente.
Compilador
Programa que convierte código fuente en código máquina.
Cookie
Archivo que almacena información sobre el usuario en un sitio web.

D

DNS
Sistema de nombres de dominio que traduce direcciones IP en nombres legibles.
Debugging
Proceso de encontrar y corregir errores en el código.
Driver
Programa que permite la comunicación entre hardware y sistema operativo.

E

Ethernet
Tecnología estándar para redes locales.
Encryption
Proceso de codificación de información para protegerla.

F

Firewall
Sistema de seguridad que controla el tráfico en una red.
Firmware
Software integrado en el hardware que controla su funcionamiento.

H

Hardware
Componentes físicos de un sistema informático.
HTML
Lenguaje de marcado para crear páginas web.

I

IP
Protocolo de Internet para identificar dispositivos en la red.
Inteligencia Artificial
Sistemas capaces de realizar tareas que requieren inteligencia humana.

L

Lenguaje de Programación
Conjunto de instrucciones para crear programas informáticos.
Linux
Sistema operativo libre y de código abierto.

M

Malware
Software diseñado para causar daño o robar información.
Monitor
Dispositivo de salida que muestra información visual.

P

Pixel
Unidad mínima de una imagen digital.
Proxy
Servidor intermediario que gestiona las peticiones entre un cliente y un servidor.

R

Router
Dispositivo que dirige el tráfico de red entre diferentes redes.
RAM
Memoria de acceso aleatorio que almacena datos temporalmente.

S

Software
Conjunto de programas y aplicaciones de un sistema informático.
Servidor
Equipo que ofrece servicios a otros dispositivos en la red.

T

TCP/IP
Conjunto de protocolos para la transmisión de datos en Internet.
Trojan
Tipo de malware que permite acceso no autorizado al sistema.

U

URL
Dirección única de una página web en Internet.
Ubuntu
Distribución de Linux basada en Debian.

V

VPN
Red privada virtual que asegura la conexión a Internet.
Virus
Programa malicioso diseñado para replicarse y dañar el sistema.
